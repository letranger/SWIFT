1 Introduction

什麼是Firebase？Firebase是由Google收購的子公司，所提供的是行動應用程式開發的平台服務，也就是BaaS(Backend as a Service)，它能夠幫助我們快速開發我們的應用程式，並且支援跨平台包括Android、iOS以及Web，幫我們整合大部分建立一個服務所需要的後台功能。主要像是提供雲端給使用者架起網頁、提供資料庫供使用者存取資料以及做後台分析等等。1

1.1 Firebase的歷史

Firebase從Envolve演變而來。 Envolve是James Tamplin和Andrew Lee在2011年創立的一家早期創業公司。Envolve為開發人員提供了一個API,該API允許將在線聊天功能集成到他們的網站中。釋放聊天服務後,它發現envlove被用來傳遞不是聊天消息的應用程序數據。開發人員使用Envolve同步應用程序,以分離實時體系結構和為其提供動力的聊天系統。2011年9月,Tamplin和Lee單獨成立了Firebase。最後於2012年4月向公眾發佈。2

3 Setting up Firebase project

3.1 Step by step

  1. 打開Firebase Console: https://console.firebase.google.com/
  2. 開啟或新增專案
  3. 進入專案後，按新增應用程式
  4. 選取平台 iOS
  5. 下載 plist 文件 並且放入 Xcode 裏面
  6. 添加PodFile
    1. 打開 Terminal
    2. 前往 Xcode 檔案夾 (xcodeproj 位置)

    3. 輸入指令: pod init

             pod init

    4. 打開 PodFile 或 輸入指令

              open Podfile

    5. 在target(iOS) 裡加入

            pod 'Firebase/Analytics'

  7. 安裝pod

       pod install
  8. 開啟xcworkspace

  9. 在xxxApp.swift加入

    1:    import Firebase

    在@main加入

    1: init() {
2:     FirebaseApp.configure()
3: }
